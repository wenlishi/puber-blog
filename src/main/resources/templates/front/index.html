欢迎访问 puber-blog

这是一个基于 Spring Boot + Thymeleaf + PostgreSQL 开发的个人博客系统。

项目初始化完成！

基础骨架代码已搭建完成，包括：

  • Maven 项目配置文件 (pom.xml)
  • Spring Boot 配置文件 (application.yml)
  • 数据库初始化脚本 (init.sql)
  • Git 配置文件 (.gitignore, .gitattributes)
  • 项目文档 (README.md)
  • 实体类、Repository、配置类、工具类等基础代码

请按照需求文档继续开发各功能模块。

快速开始

1. 初始化数据库

psql -U postgres -d puber_blog -f init.sql

2. 配置数据库连接

修改 application-dev.yml

3. 启动项目

mvn spring-boot:run
